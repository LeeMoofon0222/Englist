# gt_test_app
<h2>介紹</h2>
這是使用Flutter作出來的App，利用fire base實作登入驗證以及註冊介面，達成email登入、匿名登入，以及google登入功能。在進去頁面後。我則是串接cloud translate api與Real time storage，兩個api實現自動翻譯的功能與CRUD的功能。<br><br>在不久後會上架Play商城
